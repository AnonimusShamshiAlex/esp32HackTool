## 📈 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Как работает ARP Spoofing:

```
1. ESP32 подключается к целевой сети
2. Отправляет фальшивые ARP ответы
3. Убеждает роутер, что ESP32 = все устройства
4. Убеждает устройства, что ESP32 = роутер
5. Весь трафик уходит в никуда
6. Интернет полностью блокируется
```

### Типы атак в режиме 5:

| Атака | Количество | Цель |
|-------|------------|------|
| ARP Spoofing к роутеру | 254 IP × 3 | Обмануть роутер |
| ARP Spoofing ко всем | 100 пакетов | Обмануть устройства |
| Gratuitous ARP | 100 пакетов | Объявить себя роутером |
| ARP Cache Poisoning | 100 пакетов | Отравить кеш |
| TCP Flood | 200 подключений | Забить NAT |
| Broadcast Flood | 100 пакетов | Создать хаос |
