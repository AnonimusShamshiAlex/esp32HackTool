## 🔥 5 видов атак в режиме INFINITE ARP (режим 5):

### 1. 🎯 **ARP SPOOFING (Отравление ARP кеша)**
```
Цель: Обмануть устройства в сети
Механизм: 
- ESP32 говорит роутеру: "Я - все устройства"
- ESP32 говорит устройствам: "Я - роутер"

Результат: 
- Весь трафик уходит в никуда
- Устройства теряют интернет
```

### 2. 🌊 **TCP FLOOD (Забивание NAT таблицы)**
```
Цель: Перегрузить роутер
Механизм:
- 200 одновременных подключений к роутеру
- Роутер не успевает их обрабатывать

Результат:
- Таблица NAT переполняется
- Новые устройства не могут подключиться
- Существующие соединения рвутся
```

### 3. 📢 **BROADCAST FLOOD (Широковещательный шторм)**
```
Цель: Создать хаос в сети
Механизм:
- Отправка на 255.255.255.255 (всем устройствам)
- 10 разных портов одновременно

Результат:
- Все устройства в сети получают мусор
- Процессоры устройств перегружаются
- Сеть захлебывается от запросов
```

### 4. 💀 **GRATUITOUS ARP (Ложное объявление роутера)**
```
Цель: Объявить ESP32 роутером
Механизм:
- ESP32 отправляет "Я - роутер!" без запроса
- Все устройства верят этому

Результат:
- Все устройства пытаются отправить трафик на ESP32
- ESP32 не пересылает пакеты дальше
- Интернет умирает мгновенно
```

### 5. ☠️ **ARP CACHE POISONING (Отравить кеш всех устройств)**
```
Цель: Записать ложные MAC адреса
Механизм:
- Атакует случайные IP в сети
- Подменяет соответствие IP → MAC

Результат:
- Устройства не знают где настоящий роутер
- Связь полностью разорвана
- Нужна очистка ARP кеша для восстановления
```
